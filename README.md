# Express Notes App

Development mode:

```
npm install
npm run dev
```

Production mode:

```
npm run build
```

## New note template

```json
{
  "title": "Test note",
  "content": "Content of the test note.",
  "category": "Idea"
}
```

## Technologies

- Express
- TypeScript
